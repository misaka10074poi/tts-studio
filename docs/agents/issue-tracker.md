# Issue Tracker — GitHub Issues

Issues for this repo are tracked at https://github.com/misaka10074poi/tts-studio/issues.

Use the `gh` CLI (`/c/Program Files/GitHub CLI/gh.exe`):

```bash
gh issue create --repo misaka10074poi/tts-studio -t "Title" -b "Body" -a "@me"
gh issue list --repo misaka10074poi/tts-studio -L 20
gh issue view <number> --repo misaka10074poi/tts-studio
```
