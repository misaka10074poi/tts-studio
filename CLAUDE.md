# TTS Studio

Electron 桌面配音工作站。粘贴长文本 → 选音色 → 自动拆段 → 并发生成 → 本地输出完整音频。

## Tech Stack
Vite 5 + React 18 + MUI 5 + Tailwind CSS 3 + Zustand 4 + TypeScript 5 + Electron 42

## Commands
- `npm run dev` — Vite dev server (localhost:5173)
- `npm run build` — tsc -b && vite build
- `npm run electron:build` — 打包 exe
